Open-source alternative to ElevenReader. Paste text, a URL, or a document — it synthesizes audio using a cloned voice, locally.

## Features

- Text, URL, and document input
- Voice cloning from a short audio sample
- Local inference, no API keys required
- Persistent voice profiles
- Audio library with playback and export
